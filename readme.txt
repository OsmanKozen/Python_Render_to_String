Kodun sorunsuz çalışabilmesi için py ve html dosyaları aynı dizinde olmalıdır.

Bir django projesi açtığınızda "settings.py" dosyasını konfigüre etmeniz gerekmektedir. Bu örnekte lokal bir python dosyasında çalışıldığı için py dosyası içinde "settings" kütüphanesi import edilerek konfigüre edilmiştir.

Kullanım alanı:
Bir proje üstünden değil de lokal bir python dosyası ile html render edip email gönderme gibi işlemler yapmak için kullanılabilir.
